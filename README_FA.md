## ✨ معرفی

**هم‌نوا** یک اپلیکیشن پیام‌رسان مدرن است که برای کاربرانی طراحی شده که **به اینترنت بین‌الملل دسترسی ندارند**.

این اپ امکان ارتباط **امن، سریع و لحظه‌ای** را در شبکه‌های محدود فراهم می‌کند.

---

## 🚀 تکنولوژی‌ها

* Flutter (رابط کاربری کراس‌پلتفرم)
* PocketBase (بک‌اند + Realtime + احراز هویت)
* BLoC (مدیریت State)
* Clean Architecture
* GetIt (تزریق وابستگی)
* GoRouter (مسیریابی)

---

## 📱 امکانات

### 💬 پیام‌رسانی

* چت خصوصی
* چت گروهی

## 🏗️ معماری

این پروژه بر اساس **Clean Architecture** طراحی شده:

* لایه Presentation (UI + BLoC)
* لایه Domain (منطق برنامه)
* لایه Data (ریپازیتوری‌ها و منابع داده)
